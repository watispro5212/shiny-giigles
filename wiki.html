Nexus Glossary

Decoding the digital architecture of the Protocol ecosystem.

DEPLOYMENT // QUICK START GUIDE

> Step 1: Click the [INITIALIZE] uplink in the top menu.

> Step 2: Authorize the Nexus Protocol for your sector with standard Administrator clearance.

> Step 3: Execute /info in your server to confirm active telemetry.

> Step 4: Use /daily to claim your first allocation of Nexus credits and start building your streak.

> Step 5: Run /invite in a server to copy an OAuth link for adding Nexus elsewhere (requires DISCORD_CLIENT_ID on the host).

Cluster Alpha // Infrastructure

Network Architecture

SHARDING

Isolated bot process instances. Sharding allows the Nexus to scale across thousands of sectors without performance degradation.

LATENCY

Temporal delay in data pulses. Measured in milliseconds, lower latency ensures faster neural response times.

GATEWAY

Primary entry point for neural transmissions to the Discord API. Manages real-time event streams.

NODE

Specific processing unit or server cluster within the infrastructure where commands are executed.

Financials

Economy Protocols

Understanding the credit exchange and asset management matrix.

💳 Credit Reserves

Credits are the primary unit of exchange. Earned via labor packets (/work) or daily allocations (/daily with streak bonuses!), they are stored in encrypted vaults linked to your User ID.

🥷 The Robbery Matrix

Operatives can attempt to breach the credit vaults of others. This carries high risk—failed breaches result in significant credit loss. Shield Modules offer 24hr protection.

🎒 Inventory Storage

The shop has 8 items: Lucky Charms, XP Boosters, Shield Modules, Crate Keys, and more. Items can grant temporary buffs to breach success rates or credit yield.

📈 Streak System

Daily claims build a streak counter. Each consecutive day adds +100 CR to your daily reward, capped at +2000 CR. Missing two days resets your streak to 1.

🎰 Casino Matrix

Slots and Blackjack offer high-risk credit multiplication. Slots support Lucky Charm bonuses. The jackpot (3x 🎰) pays 50× your wager.

🧬 Level Scaling

Work pay scales with your Neural Level. Higher levels earn better gigs with a bonus of Level × 25 CR per job. Invest in XP Boosters for faster progression.

Security Architecture

Multi-layered defense system protecting all Nexus operations.

🚫

Blacklist Gate

Severed operatives are permanently blocked from executing any command across all sectors.

🏛️

Guild-Only Lock

All slash commands are restricted to server channels. DM execution is rejected at the protocol level.

🔐

Owner Gate

Root-level commands are hardcoded to a specific user ID. Unauthorized access returns ACCESS_DENIED.

⏳

Cooldown Engine

Per-command, per-user rate limiting with category-based durations. Owner bypasses all cooldowns.

🔇

Anti-Spam Shield

Message-level flood detection auto-deletes rapid transmissions. Memory auto-prunes every 60 seconds.

🛡️

Module Toggles

Server admins can disable entire command categories (Economy, Casino, Fun, Leveling) per-server.

🧹

Word Filter

Configurable per-server bad word list. Flagged messages are auto-deleted with a warning embed.

🔄

Error Isolation

Double try-catch pattern prevents any single command crash from killing the entire bot process.
