Privacy Protocol

VERSION 1.0.4 | LAST REVISION: MARCH 2026

01 Data Collection Parameters

NexusBot collects minimal metadata required for operation. This includes User IDs, Server IDs, and Channel IDs. We do not log content of private messages or non-command related interactions unless specific security modules (e.g. Auto-Mod) are enabled by node administrators.

02 Economy & Transaction Data

All credit balances, inventory records, and transaction histories are stored in encrypted database clusters. This data is used solely for the internal economy system and is never shared with third-party intelligence entities.

03 User Rights

Operatives may request a full data purge at any time by contacting the Support Node or using the command interface (where applicable). Removing the bot from a server does not automatically delete economy data; please contact support for absolute severance.

04 Security Standards

We utilize industry-standard encryption for all data packets. However, no digital uplink is 100% secure. Use the protocol at your own discretion within the Nexus network.
